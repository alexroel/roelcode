# Tipos de datos básicos en Python

Python maneja diferentes tipos de datos automáticamente. En esta lección aprenderemos los tipos más importantes y cómo trabajar con ellos.

## 🔢 Números

Python tiene varios tipos de números:

### Enteros (int)
```python
edad = 25
poblacion = 1000000
temperatura = -5
```

### Decimales (float)
```python
precio = 29.99
altura = 1.75
pi = 3.14159
```

### Operaciones básicas
```python
suma = 10 + 5        # 15
resta = 10 - 3       # 7
multiplicacion = 4 * 5   # 20
division = 10 / 3    # 3.333...
division_entera = 10 // 3  # 3
potencia = 2 ** 3    # 8
modulo = 10 % 3      # 1
```

## 📝 Texto (Strings)

Los strings son secuencias de caracteres:

```python
nombre = "Ana"
apellido = 'García'
mensaje = "¡Hola, mundo!"
```

### Strings multilínea
```python
poema = """
Roses are red,
Violets are blue,
Python is awesome,
And so are you!
"""
```

### Operaciones con strings
```python
nombre_completo = nombre + " " + apellido  # "Ana García"
saludo = f"Hola, {nombre}!"  # "Hola, Ana!" (f-strings)
grito = mensaje.upper()      # "¡HOLA, MUNDO!"
longitud = len(mensaje)      # 13
```

## ✅❌ Booleanos (bool)

Los booleanos solo tienen dos valores:

```python
es_estudiante = True
tiene_trabajo = False
es_mayor_edad = edad >= 18
```

### Operaciones lógicas
```python
# AND (y)
puede_votar = es_mayor_edad and es_ciudadano

# OR (o)
tiene_descuento = es_estudiante or es_jubilado

# NOT (no)
no_es_estudiante = not es_estudiante
```

## 📦 Listas

Las listas almacenan múltiples elementos:

```python
frutas = ["manzana", "banana", "naranja"]
numeros = [1, 2, 3, 4, 5]
mixta = ["Python", 3.8, True, 42]
```

### Trabajando con listas
```python
primera_fruta = frutas[0]     # "manzana"
frutas.append("uva")          # Agregar elemento
cantidad = len(frutas)        # 4
```

## 🗂️ Diccionarios

Los diccionarios almacenan pares clave-valor:

```python
persona = {
    "nombre": "Carlos",
    "edad": 30,
    "ciudad": "Madrid",
    "programador": True
}

# Acceder a valores
nombre_persona = persona["nombre"]    # "Carlos"
edad_persona = persona.get("edad")    # 30
```

## 🔍 Función type()

Para conocer el tipo de una variable:

```python
numero = 42
print(type(numero))        # <class 'int'>

texto = "Hola"
print(type(texto))         # <class 'str'>

lista = [1, 2, 3]
print(type(lista))         # <class 'list'>
```

## 🔄 Conversión de tipos

Python puede convertir entre tipos:

```python
# String a número
edad_texto = "25"
edad_numero = int(edad_texto)    # 25

# Número a string
precio_numero = 29.99
precio_texto = str(precio_numero)  # "29.99"

# A booleano
verdadero = bool(1)              # True
falso = bool(0)                  # False
```

## 💡 Tipo dinámico

Python es **dinámicamente tipado**:

```python
variable = 42         # int
print(type(variable)) # <class 'int'>

variable = "Hola"     # str
print(type(variable)) # <class 'str'>

variable = [1, 2, 3]  # list
print(type(variable)) # <class 'list'>
```

## 📚 Resumen

- **int**: Números enteros
- **float**: Números decimales  
- **str**: Texto/cadenas
- **bool**: Verdadero/Falso
- **list**: Listas de elementos
- **dict**: Diccionarios clave-valor
