# Variables en Python

Las variables son fundamentales en cualquier lenguaje de programación. En Python, son especialmente fáciles de usar.

## 🎯 ¿Qué es una Variable?

Una variable es un **contenedor** que almacena datos que pueden cambiar durante la ejecución del programa.

```python
# Declarar una variable
nombre = "Alex"
edad = 25
altura = 1.75
es_estudiante = True
```

## 📝 Reglas para Nombres de Variables

### ✅ Permitido:
- `nombre`
- `edad_usuario`
- `precio2`
- `_variable_privada`

### ❌ No permitido:
- `2nombre` (no puede empezar con número)
- `mi-variable` (no se permiten guiones)
- `class` (palabra reservada)

## 🏷️ Convenciones de Nombrado

Python usa **snake_case**:

```python
# Buenas prácticas
nombre_completo = "Alex Roel"
numero_de_telefono = "123456789"
es_mayor_de_edad = True

# Evitar
nombreCompleto = "Alex Roel"  # camelCase
NombreCompleto = "Alex Roel"  # PascalCase
```

## 🔄 Reasignación de Variables

```python
mensaje = "Hola"
print(mensaje)  # Hola

mensaje = "Adiós"
print(mensaje)  # Adiós

# Python es dinámicamente tipado
numero = 42
numero = "Ahora soy texto"  # ¡Válido!
```

## 📚 Tipos de Datos Básicos

```python
# Números enteros
edad = 25

# Números decimales
precio = 29.99

# Texto (strings)
nombre = "Python"

# Booleanos
es_activo = True
esta_terminado = False
```

## 💡 Consejos

1. **Usa nombres descriptivos**: `precio_producto` mejor que `p`
2. **Sé consistente** con el estilo de nombrado
3. **Evita palabras reservadas** como `class`, `def`, `if`

¡Practica creando tus propias variables!
