# Tu primer programa en Python

¡Es hora de escribir tu primer programa en Python! En esta lección aprenderemos los conceptos básicos y ejecutaremos nuestro primer código.

## 🚀 Hello, World!

Empecemos con el programa más famoso en programación:

```python
print("¡Hola, mundo!")
```

### ¿Qué hace este código?

- `print()` es una **función** que muestra texto en la pantalla
- Las comillas (`"`) indican que estamos trabajando con texto (string)
- `¡Hola, mundo!` es el mensaje que queremos mostrar

## 💻 Cómo ejecutar tu programa

### Opción 1: Terminal/Consola
```bash
python mi_programa.py
```

### Opción 2: VS Code
1. Abre el archivo `.py`
2. Presiona `F5` o usa el botón "Run"
3. Ve el resultado en la terminal integrada

## 📝 Estructura básica de un programa Python

```python
# Esto es un comentario
print("Mi primer programa")  # Comentario al final de línea

# Las líneas vacías están permitidas

print("Segunda línea")
```

### Elementos importantes:

1. **Comentarios**: Empiezan con `#`
2. **Indentación**: Python usa espacios para organizar el código
3. **Funciones**: Como `print()` para mostrar texto

## 🎯 Ejercicio práctico

Crea un archivo llamado `mi_primer_programa.py` y escribe:

```python
# Mi primer programa en Python
print("¡Hola! Soy [tu nombre]")
print("Estoy aprendiendo Python")
print("¡Este es mi primer programa!")
```

## 💡 Consejos importantes

- **Guarda siempre** tu archivo con extensión `.py`
- **Usa nombres descriptivos** para tus archivos
- **Practica escribiendo** el código, no solo copiando

## 🐍 ¿Sabías qué?

Python fue diseñado para ser **fácil de leer**. ¡Por eso es perfecto para principiantes!
