# ¿Qué es Java?

Java es uno de los lenguajes de programación más populares y utilizados en el mundo empresarial. En esta lección descubriremos sus características y ventajas.

## ☕ Historia de Java

Java fue desarrollado por **James Gosling** en Sun Microsystems en 1995. Su lema principal es **"Write Once, Run Anywhere"** (Escribe una vez, ejecuta en cualquier lugar).

### Características principales:

- **Orientado a objetos**: Todo en Java son objetos
- **Multiplataforma**: Funciona en cualquier sistema operativo
- **Robusto**: Manejo automático de memoria
- **Seguro**: Sistema de seguridad integrado

## 🚀 ¿Por qué Java?

Java es ideal para:

1. **Desarrollo empresarial**: Aplicaciones corporativas robustas
2. **Aplicaciones web**: Backend con frameworks como Spring
3. **Desarrollo móvil**: Aplicaciones Android nativas
4. **Big Data**: Herramientas como Hadoop y Apache Spark

## 🏗️ Arquitectura de Java

Java funciona con:

- **JVM (Java Virtual Machine)**: Ejecuta el código Java
- **JRE (Java Runtime Environment)**: Entorno de ejecución
- **JDK (Java Development Kit)**: Kit de desarrollo completo

## 💼 Oportunidades Laborales

Java es uno de los lenguajes más demandados:

- Desarrollador Java Backend
- Desarrollador de aplicaciones empresariales
- Desarrollador Android
- Arquitecto de software
