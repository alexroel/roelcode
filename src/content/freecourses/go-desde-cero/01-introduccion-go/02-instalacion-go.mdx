# Instalación de Go

En esta lección aprenderemos cómo instalar Go en diferentes sistemas operativos y configurar nuestro entorno de desarrollo.

## Instalación en Windows

1. Descarga el instalador desde golang.org
2. Ejecuta el archivo .msi
3. Sigue las instrucciones del instalador
4. Verifica la instalación

## Instalación en macOS

### Usando el instalador oficial
```bash
# Descarga el .pkg y ejecuta
```

### Usando Homebrew
```bash
brew install go
```

## Instalación en Linux

### Ubuntu/Debian
```bash
sudo apt update
sudo apt install golang-go
```

### CentOS/RHEL
```bash
sudo yum install golang
```

## Verificación de la instalación

```bash
go version
```

## Configuración del entorno

- GOROOT: Directorio donde está instalado Go
- GOPATH: Directorio de trabajo
- GOBIN: Directorio de ejecutables
