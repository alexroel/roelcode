# Tu primer programa en Go

¡Es hora de escribir tu primer programa en Go! Comenzaremos con el tradicional "Hola Mundo".

## Estructura básica

```go
package main

import "fmt"

func main() {
    fmt.Println("¡Hola, mundo!")
}
```

## Explicación del código

- `package main`: Define el paquete principal
- `import "fmt"`: Importa el paquete para formateo
- `func main()`: Función principal del programa
- `fmt.Println()`: Imprime texto en la consola

## Ejecutar el programa

```bash
# Opción 1: go run
go run main.go

# Opción 2: compilar y ejecutar
go build main.go
./main
```

## Conceptos importantes

- Todo programa Go debe tener un paquete `main`
- La función `main()` es el punto de entrada
- Go es un lenguaje compilado
- No necesitas punto y coma al final de línea
