## Programación Orientada a Objetos

Los paradigmas de programación que se usa con este lenguaje de programación y fue echo para este paradigma de programación orientada a objeto. La Programación Orientada a Objetos (POO) nos ayuda a desarrollar sistemas más ordenado y estructura. Java surgió con este paradigma y es uno de los lenguajes que define el rumbo que sigue la POO.

Cuatro pilares de Programación Orientada a Objetos

- Encapsulamiento
- Abstracción
- Herencia
- Polimorfismo
